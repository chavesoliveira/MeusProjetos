Conceito

Segundo o PMBOK, um projeto é um esforço temporário empreendido para criar um produto, serviço ou resultado exclusivo. ... Lembre-se que ser “temporário” significa que os projetos devem ter um início e um término definidos; não significa de curta duração.

Entretanto num mundo onde o desenvolvimento web está cada vez mais procurado e dominante no que tange informatização, esse projeto terá uma idéia de treiner e dev web baby, uma forma de adquirir mais conhecimentos e colocá-los em prática durante todo me curso.
