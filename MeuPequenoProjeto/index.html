PROJETO VIVA

Viva Se Alimente Faça Exercícios

Segundo o PMBOK, um projeto é um esforço temporário empreendido para criar um produto, serviço ou resultado exclusivo. ... Lembre-se que ser “temporário” significa que os projetos devem ter um início e um término definidos, o que não significa ser de curta duração.

Entretanto num mundo onde o desenvolvimento está cada vez mais atenuado e dominante, esse projeto terá uma idéia de mudança, uma forma de adquirir conhecimentos e colocá-los em prática durante todo um percurso da vida.

Planejar um projeto é importante para todos que valorizam boas práticas de gestão e entende que a definição de um escopo de trabalho tem grande relevância para a projeção de uma imagem estratégica. Apesar de parecer uma tarefa extremamente complicada, com dedicação e técnica, a etapa do planejamento pode ser muito mais fácil do que se imagina.

Por sua vez, ela começa com o checklist, um trabalho muito comum que se repete pelo cotidiano das pessoas em geral. O ato de conferir listas é um importante aliado para o cumprimento das etapas.

Afinal, o mais fundamental de planejar um projeto é conferir sempre o que falta para ele ser concluído. O que não estiver ocorrendo de acordo com o planejado deverá ser anotado em uma lista diária.

Podemos utilizar um checklist como um ponto de partida para abordar alguns passos para quem deseja transformar o planejamento em uma referência para novos gerentes e um lembrete para os colaboradores mais experientes. É recomendável verificar essas dicas constantemente para se certificar de que nenhuma etapa importante foi perdida.

Se atentarmos com destreza para cada nível desse processo, iremos sair com um grande planejamento e obteremos sucesso no final de cada etapa.

Note que para cada situação aparecem questionamentos pertinentes que devemos esclarecer para que não haja falhas no projeto.
